# QR Code Component Challenge

A pixel-perfect implementation of a front-end layout challenge, focusing on semantic HTML5 and CSS3 structure.

## 🎯 Project Overview
This component highlights layout control, centering techniques, and responsive styling.

### Live Preview
* Repository: `htmlandcss`
* Project: Frontend Mentor Card Style

---

## 🛠️ Tech Stack & Key Concepts Used
* **Semantic HTML5:** `<main>`, `<article>`, `<header>` for accessibility.
* **CSS3:** Flexbox for centering, `box-sizing: border-box`, and custom `box-shadow` usage.
* **Version Control:** Git/GitHub workflows.

---

## 🧠 Key Learnings
1. **Layout Strategy:** Prioritized padding and relative sizing over fixed heights to prevent content overflow.
2. **Structural Integrity:** Utilized semantic HTML instead of `<div>` tags for better accessibility.
